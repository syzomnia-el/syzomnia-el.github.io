osu!searcher 使用教程

osu!searcher 是一个为 osu! 开发的本地谱面查找工具，无需启动 osu! 即可快速查找已下载的谱面。

功能​

  • 查看本地谱面

  • 按关键字筛选谱面

  • 检查重复的谱面

  • 对特定条件，如谱面编号、歌名、艺术家、谱师等，按关键字筛选谱面

要求​

  • Python 3.6 或更高版本
  • （GUI 版）PySide 6

下载​

你可以从位于 Github 的项目仓库中下载 osu!searcher 的源代码。
项目地址：https://github.com/syzomnia-el/osu-searcher

  • 你可以使用 git 将项目仓库复制到本地：

    • CLI 版：

      git clone https://github.com/syzomnia-el/osu-searcher.git

    • GUI 版：

      git clone https://github.com/syzomnia-el/osu-searcher.git -b gui

  • 也可以通过下载 zip 压缩包来获取。

使用​

  1. （如果使用 git 获取，请忽略此步骤）将 zip 文件解压到适当的位置。
  2. 打开 osu!searcher 的所在目录。
  3. 运行 startup.cmd（或 startup.sh）脚本。
  4. 首次使用时，需要先输入谱面存储目录的绝对路径。

命令​

命令描述
check检查重复的谱面
exit退出 osu!searcher
find <keyword>按关键词筛选谱面
flush刷新谱面信息缓存
list查看所有本地谱面
path修改谱面存储路径
