能再多谈谈自己吗？​

当然可以。我目前是软件工程专业在读本科生，即将面临毕业。
最熟悉的语言是 Java ，也会一些 C/C++ 和 Python 等其他语言，偶尔会完全基于自己的需求去写一些小工具。

关于头像​

我现在的头像来源于 Waifu Labs v2，并在 AI 生成图像的基础上做了一些修改，以使其更加令人满意。

Waifu Labs 项目由来自 MIT 的 Sizigi Studios 团队研究开发。只需4个简单的步骤，AI 就能生成质量相当不错的、符合你喜好风格的二次元人物头像。
